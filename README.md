# Opera-Dance
Opera Dance Studio is a portfolio website created to present a dance studio’s classes, programs, and atmosphere through an elegant and engaging design. The project was developed using HTML, CSS, and Bootstrap to ensure a responsive layout and smooth navigation across devices.

The website highlights key sections such as dance classes, schedules, gallery, and contact information, demonstrating front‑end development skills and the ability to create visually appealing and well‑structured websites.
